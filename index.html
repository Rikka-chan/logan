Logan Log Viewer

logan is a web based interface or log viewing/searching using tail/head and grep Built upon:

  • Python 2.6
  • Flask - http://flask.pocoo.org/
  • Pytailer - http://code.google.com/p/pytailer/
  • Grin - http://pypi.python.org/pypi/grin
